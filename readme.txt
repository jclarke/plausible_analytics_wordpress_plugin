This plugin adds the Plausible Analytics tracking code to the header of your wordpress blog.

== Description ==
This plugin automatically adds the Plausible Analytics javascript code to the header of your wordpress blog. You can optionally change the tracking domain in the settings page. You can also optionally disable tracking of logged in admin users.

== Installation ==
Upload the plausible_unofficial folder to wp-content/plugins

Activate the plugin and verify your domain is correct in Settings->Plausible Analytics Settings.
